# gen_comment_block

コメントのブロックを生成するPythonのスクリプトです
<br>
例
```
$ ./gen_comment_block.py --comment hello
#----------------------------------------------------------------------------------------------------#
#                                                                                                    #
#                                               hello                                                #
#                                                                                                    #
#----------------------------------------------------------------------------------------------------#
```

# install
好きな場所にクローンしてパスを通すだけです。
```
git clone https://github.com/sei-12/gen_comment_block.git
```
必要なのは`#!/usr/bin/python3`です
標準ライブラリしか使用していません
